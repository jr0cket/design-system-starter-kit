Welcome to the Hacked Design System Starter Kit

A rapid prototyping environment using the Lightning Design System, brought to you by Salesforce UX.

Get Started

Edit files in the src directory to start building your prototype.

Start by looking at the file src/views/index.html file, which contains the html that defines this page you are reading right now.

Lightning Design System component documentation and markup available at: https://www.lightningdesignsystem.com.

Example Styles to try

Here are some example stlyes to apply to the page

Titles / Headings

Use the class slds-text-heading--small to define a small heading, or slds-text-heading--medium for a medium heading or slds-text-heading--large for a large heading.

To affect the spacing (margin) around a title, also add the slds-m-top--medium to define a medium margin around the heading, changing medium for small or large for diferent sizes.

In the **Components - Utilities** section there are a range of different spacing styles available

Paragraphs

Use the class slds-text-body--regular to define a regular sized (13px) paragraph of text, or slds-text-body--small for a small font for the paragraph, a more pale-colored 12px font.

To affect the spacing (margin) around a paragraph, also add the slds-m-top--medium to define a medium margin around the heading, changing medium for small or large for diferent sizes.
